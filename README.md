## SkillCraft Technology - Data Science Internship

### Task 02: Exploratory Data Analysis (EDA) on Titanic Dataset

### 📖 Project Overview
Performed Exploratory Data Analysis (EDA) on the Titanic dataset to understand passenger characteristics and identify the key factors that influenced survival.

### 🔄 Workflow
- Imported the dataset
- Explored the data
- Cleaned missing values
- Performed statistical analysis
- Created visualizations
- Extracted key insights

### 📊 Visualizations
- Survival Distribution
- Age Distribution
- Passenger Class vs Survival
- Gender vs Survival
- Age vs Survival
- Correlation Heatmap

### 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Git & GitHub

### 💡 Key Insights
- Female passengers had a higher survival rate.
- First-class passengers were more likely to survive.
- Age and passenger class influenced survival.
- Most passengers traveled in Third Class.

### ✅ Conclusion
Successfully performed data cleaning, visualization, and exploratory data analysis to uncover meaningful insights from the Titanic dataset.
